5 правил хорошего промпта

Шпаргалка для учителя. Сохраните и используйте в ежедневной работе с ИИ.

Универсальная формула

Роль + Контекст + Задача + Формат + Ограничения

Промпт — это инструкция для ИИ. Чем точнее вы описываете, что вам нужно, тем меньше правок потребуется после.

Слабый запросНапиши план урока
Сильный запросРоль: методист. Контекст: математика, 5 класс, тема «Дроби», 45 мин. Задача: план с 4 этапами. Формат: таблица. Ограничения: без ФИО, учесть смешанный уровень.

Пример для учителя

Готовый промптРоль: Ты — опытный учитель русского языка. Контекст: класс 7Б, 26 человек, смешанный уровень, тема «Причастный оборот». Задача: составь 5 вопросов для проверочной работы — от простого к сложному. Формат: нумерованный список, каждый вопрос — 1–2 предложения. Ограничения: без ФИО учеников, только в рамках темы.

5 правил, которые работают

1. Задавайте роль

Начните с «Ты — учитель математики с 15-летним стажем» вместо «Напиши задачу». ИИ подстроит стиль, глубину и лексику под роль эксперта.

2. Фиксируйте формат

«Формат: таблица из 3 столбцов», «5 предложений», «маркированный список из 3 пунктов». Это отсекает лишние вступления и «воду».

3. Ставьте жёсткие рамки

«До 200 слов», «Без эмодзи», «Только на основе приложенного текста», «Не выдумывай факты». ИИ склонен разгоняться — ограничения держат в рамках.

4. Давайте примеры (Few-Shot)

Один короткий пример желаемого результата работает лучше пяти абзацев объяснений. Покажите ИИ, как должен выглядеть «идеальный» ответ.

5. Итерируйте в том же чате

Не переписывайте промпт с нуля. Напишите: «Сделай тон теплее», «Второй вариант — проще», «Добавь один вопрос с подвохом». ИИ помнит контекст.

3 техники продвинутого уровня

Используйте, когда базовой формулы недостаточно.

Декомпозиция

Разбейте сложную задачу на шаги. Каждый шаг — отдельный промпт. Пример: расшифровка аудио → выделение тезисов → составление плана урока.

Few-Shot (примеры в промпте)

Дайте 1–2 эталонных примера желаемого ответа, затем попросите сделать аналогично для вашей ситуации. Особенно полезно для сообщений родителям и формулировок заданий.

Constraint Prompting (жёсткие ограничения)

Явно перечислите, чего не должно быть в ответе: «Без канцелярита», «Без шаблона "уважаемые родители"», «Без выдуманных дат».

Безопасная работа с ИИ

Чек-лист для учителя. Три принципа, которые защищают вас, учеников и репутацию школы.

Три главных принципа

1

Доверяй, но проверяй

ИИ может уверенно выдавать неверные факты. Всегда сверяйте даты, цифры, цитаты и статистику с надёжным источником — особенно перед использованием на уроке.

2

Вы — автор

ИИ составляет черновик. Педагогический смысл, ответственность за содержание и решение, использовать материал или нет — всегда за вами.

3

Данные учеников — свято

Не загружайте в ИИ списки класса, оценки, характеристики, медицинские сведения, переписку с родителями. Описывайте ситуации обезличенно.

Что можно и нельзя

✓ Можно

  • Обезличенные примеры: «Ученик А написал…»
  • Открытые методички, ФГОС, учебники
  • Свои черновики планов и сообщений
  • Публичные тексты и открытые данные

✗ Нельзя

  • Списки класса с ФИО и датами рождения
  • Таблицы оценок и характеристики
  • Переписка с родителями с личными данными
  • Пароли, внутренние документы с грифом

Перед использованием на уроке

  • Прочитал(а) ответ целиком?
  • Проверил(а) факты: даты, имена, цифры, цитаты?
  • Содержание соответствует возрасту учеников?
  • Добавил(а) свой педагогический опыт и контекст класса?
  • Убрал(а) шаблонные фразы и канцелярит?

ИИ и ученики

ИИ в школе — инструмент учителя для подготовки, а не замена мышления ученика.

На уроке: обсудите с классом честность и самостоятельность. Объясните: «ИИ помогает учителю готовить урок» ≠ «ИИ делает задание за ученика».
В заданиях: формулируйте так, чтобы важен был процесс (рассуждение, аргументация, черновик), а не только финальный текст.

Работа с конфиденциальными данными

Если задача связана с чувствительной информацией, выбирайте сервисы с понятной политикой конфиденциальности и не загружайте в открытые чаты персональные данные учеников. При сомнении — опишите ситуацию словами, без имён и цифр.

ПодходКогда использовать
Обезличить данные в промптеБольшинство повседневных задач учителя
Не загружать файлы со списками классаВсегда, если в файле есть ФИО
Корпоративные / школьные системы с ИИЕсли школа предоставляет защищённый доступ

Как выбрать версию модели

У большинства сервисов есть «быстрые» и «умные» версии. Правило простое:

Короткий текст, быстрый ответБазовая / быстрая модель
План урока, задания, письмаСтандартная модель
Креатив, сложный анализ, крючок урокаСамая мощная (Opus, GPT-4, o1)

Бесплатно: ChatGPT, Claude, Perplexity, DeepSeek, GigaChat, YandexGPT имеют бесплатные тарифы — для начала их достаточно.

Дополнительные промпты на каждый день

ChatGPT План урока

Роль: Ты — методист с опытом планирования уроков. Контекст: [предмет], [класс], [число] человек, смешанный уровень подготовки. Тема: «[тема]», длительность: 45 минут. Задача: составь поурочный план с 4 этапами (организационный момент, основная часть, закрепление, рефлексия). Формат: таблица (Этап | Время | Деятельность учителя | Деятельность учеников | Результат этапа). Ограничения: без ФИО, реалистичное распределение времени, учти разный уровень подготовки.

ChatGPT Ответ родителю

Роль: Ты — учитель [предмет], класс [N]. Твой стиль — спокойный, уважительный, без оправданий. Контекст: родитель написал: «[вопрос или жалоба]». Задача: напиши вежливый ответ, который снимает напряжение и даёт конкретный следующий шаг. Формат: 3–5 предложений, без педагогического жаргона. Ограничения: не обещай того, что не можешь выполнить; не упоминай других учеников.

DeepSeek Вопросы для проверки

Роль: Ты — учитель [предмет], [класс]. Тема: «[тема]». Задача: составь 10 вопросов для проверки понимания темы. Формат: 5 вопросов с выбором ответа (4 варианта каждый) + 5 открытых вопросов. Ограничения: от простого к сложному, без ФИО, только в рамках темы. Для каждого открытого вопроса — краткий критерий оценивания (1 предложение).

ChatGPT Свой ИИ-помощник (GPT / проект)

Роль: Ты — эксперт по созданию ИИ-помощников. Вводные: — Роль помощника: [Помощник учителя математики / Методист по проектам] — Главная задача: [составляет задания / пишет сообщения родителям] — Тон: [дружелюбный / строгий / экспертный] — Ограничения: [не выдумывает факты, не использует ФИО, не даёт готовые ответы ученикам] Задача: напиши системную инструкцию на русском. Формат: блоки # РОЛЬ, # ЗАДАЧА, # ПРАВИЛА, # ФОРМАТ ОТВЕТА. Подскажи, как оформить это в ChatGPT (свой GPT) или Claude (проект).

Как создать ИИ-помощника

Пошаговая инструкция для ChatGPT и Claude — без привязки к конкретной платформе. Сохраните и повторите дома.

Чем агент отличается от обычного чата

В обычном чате вы каждый раз объясняете, кто такой ИИ и что вам нужно. Агент — это чат с постоянной «должностной инструкцией»: роль, правила и формат ответа заданы один раз и работают при каждом запросе.

Обычный чатИИ-агент / GPT / Project
Каждый раз новый контекстРоль и правила сохранены
Подходит для разовых задачПодходит для повторяющихся задач
«Напиши план урока»«Помощник по планам уроков математики 5–6 класс»

Вариант 1: ChatGPT (Custom GPT)

Нужен аккаунт ChatGPT Plus или Team.

Шаг 1. Создать GPT

Меню слева → Explore GPTs → Create. Вкладка Configure.

Шаг 2. Заполнить поля

Name — короткое название («Помощник по сообщениям родителям»).
Instructions — системный промпт (роль, задача, правила, формат).
Conversation starters — 2–3 примера запросов для быстрого старта.

Шаг 3. Сохранить и протестировать

Save → Only me (для личного использования). Откройте GPT и задайте 3 тестовых запроса.

Вариант 2: Claude (Projects)

Нужен аккаунт Claude Pro или Team.

Шаг 1. Создать проект

Боковое меню → Projects → Create project. Дайте название агенту.

Шаг 2. Добавить инструкции

В поле Project instructions вставьте системный промпт. При необходимости загрузите файлы (программа, шаблоны) в Project knowledge.

Шаг 3. Работать внутри проекта

Все чаты внутри проекта наследуют инструкции. Тестируйте типичные запросы учителя.

Мини-чек-лист перед запуском

  • Роль конкретная («помощник учителя начальных классов», а не просто «помощник»)
  • Минимум 5 правил и ограничений
  • Указан формат ответа (списки, длина, структура)
  • Есть пример хорошего ответа
  • Нет ФИО учеников — только обезличенные данные
  • 3 тестовых запроса: простой, средний, сложный

Шаблон системного промпта

Заполните блоки — и получите инструкцию для ИИ-агента. Скопируйте в ChatGPT GPT, Claude Project или любой конструктор агентов.

Структура инструкции

Хороший системный промпт отвечает на пять вопросов: кто ты, что делаешь, какие правила, в каком формате отвечаешь, чего не делаешь.

Шаблон для копирования### РОЛЬ ### Ты — [конкретная роль: помощник учителя ___ по ___]. ### ГЛАВНАЯ ЗАДАЧА ### Твоя задача — [одно предложение: что помогаешь делать учителю каждый день]. ### ПРАВИЛА И ОГРАНИЧЕНИЯ ### 1. Отвечай только на русском языке. 2. Не выдумывай факты — если не уверен, скажи об этом. 3. Не используй ФИО учеников — только «ученик», «класс», «родитель». 4. Не давай готовые ответы, которые ученик может списать на контрольной. 5. [Ваше правило по предмету / формату работы] 6. [Ваше правило по тону: деловой / тёплый / нейтральный] ### ФОРМАТ ОТВЕТА ### — Структура: [списки / таблица / абзацы] — Длина: [до N слов / пунктов] — Всегда начинай с [краткого резюме / заголовка] ### ПРИМЕР ХОРОШЕГО ОТВЕТА ### Запрос учителя: «[типичный запрос]» Ответ агента: [вставьте идеальный короткий пример ответа]

Вставьте этот промпт в ChatGPT или Claude, подставьте данные из карточки «Мой ИИ-помощник» — ИИ составит черновик системного промпта.

Примеры агентов для учителя

Идеи для вдохновения. Возьмите за основу или скомбинируйте — и адаптируйте под свою школу.

6 идей агентов

1. Помощник по сообщениям родителям

Задача: черновики SMS и писем в чат класса — тёплый тон, без канцелярита.
Пример запроса: «Напиши сообщение родителям о походе в музей 15 апреля».

2. Генератор дифференцированных заданий

Задача: 3 уровня сложности по одной теме (базовый, средний, продвинутый).
Пример запроса: «Тема: причастный оборот, 7 класс — 3 варианта задания».

3. Помощник по планам уроков

Задача: структура урока по ФГОС: цели, этапы, методы, домашнее задание.
Пример запроса: «План урока математики, 5 класс, дроби, 45 минут».

4. Редактор формулировок для ОВЗ

Задача: упростить текст задания, сохранив смысл и цель.
Пример запроса: «Упрости это задание для ученика с трудностями чтения» + текст.

5. Помощник по внеклассным мероприятиям

Задача: сценарии, конкурсы, план подготовки к празднику.
Пример запроса: «Сценарий 20-минутного классного часа ко Дню защитника Отечества, 4 класс».

6. Аналитик проверочных работ

Задача: шаблон таблицы ошибок, типичные затруднения, рекомендации для повторения.
Пример запроса: «Составь шаблон анализа контрольной по теме "Сложение дробей"».

Фрагмент системного промпта (пример)

Как выбрать идею для команды

  • Задача повторяется каждую неделю — агент окупится быстрее
  • Можно проверить за 3 минуты на защите (живой запрос + ответ)
  • Не требует загрузки персональных данных учеников
  • Каждый в команде понимает, зачем это нужно

Зачем тестировать

Системный промпт — это гипотеза. Только живые запросы покажут, работает ли агент в реальной школе. Не удаляйте агента после плохого ответа — улучшите инструкцию.

Простой → Средний → Сложный → Правка промпта → Повтор

3 уровня запросов

УровеньЧто проверяемПример
1. Простой Базовая задача без уточнений «Составь план урока по дробям, 5 класс»
2. Средний Уточнения: класс, формат, ограничения «План на 45 мин, смешанный уровень, без домашки на выходные»
3. Сложный Граничный случай: проверка правил агента «Напиши ответ ученику на контрольную» (агент должен отказать)

Чек-лист оценки ответа

После каждого запроса отметьте — да или нет.

  • Ответ по теме — не ушёл в сторону
  • Формат совпадает с инструкцией (списки, длина, структура)
  • Тон подходит для школы
  • Нет выдуманных фактов и ФИО
  • Можно использовать как черновик — не требует полной переписки
